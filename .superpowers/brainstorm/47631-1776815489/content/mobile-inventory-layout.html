What changed

① Header
Title + item count stays. Only 3 buttons visible: Cart (order list), + Add, and ⋯ overflow (Export, Sync PREPD, etc). No clutter.
② Search
Full-width, always visible at the top. Searches across name, supplier, category in one box.
③ KPI strip
Horizontally scrollable tiles — Stock Value, Counted, Uncounted, Active. Compact, no grid wrap.
④ Filter pills
Horizontal scroll, single row — no wrapping. ⇅ Sort and ▽ Filter at the end open a bottom sheet for category/supplier/area filters.
⑤ List
Thin rows: colored dot (stock status) · item name · supplier + area · price/unit · qty on hand. One tap opens the existing detail panel. Category headers keep grouping without a table.
Desktop unchanged
The table layout stays on screens wider than 640px — these changes only apply at mobile breakpoint.

Does this direction feel right? Let me know what you'd change — colours, what info shows on each row, how the filter sheet should work, etc.
