Count Mode — Item Row Design

Currently the category badge + location text sit on the same line as the item name — they collide on narrow screens. Pick a layout that fixes this.

A

Two-line rows — category as subtitle

Item name on top. Category + location as a small coloured dot + text below. Counted rows show qty + variance on the right. Expanded row has the stepper input. Clean and readable at any name length.
