Revised Design — Structure vs. Count

Product structure (fixed) tells the system the physics of the item. Count unit (flexible) is just a view on that structure.

Core principle: Structure is fixed. Count unit is a lens.

Product structure defines the physics: case → packs → items → weight. This never changes.

Count unit is just how staff look at it during a session — case, pack, each, kg. All are valid lenses on the same structure.

Recipes always cost in base units (g / ml / each), derived from structure alone, independent of how staff count.
